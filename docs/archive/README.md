## Why it's preserved

The Blue Ridge Ruby 2026 workshop concluded 2026-05-02. These docs aren't part of the active project surface anymore, but they're still useful as:

- attribution and history for the workshop attendees,
- context for ADRs (especially [ADR-0010](../adr/0010-charm-ruby-tui-framework.md) and [ADR-0013](../adr/0013-revert-to-rooibos.md), which reference the workshop framing),
- a worked example of the staged-plan / Claude-Code collaboration shape from `claude-code-prompt.md`.

For the current state of the project, see [`../index.md`](../index.md), [`../ROADMAP.md`](../ROADMAP.md), and the active ADRs in [`../adr/`](../adr/).
